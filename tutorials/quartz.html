Job scheduling with Quartz

1. Introduction

Easy Batch engine implements the java.util.concurrent.Callable interface so it can be easily scheduled using the java.util.concurrent.ScheduledExecutorService API. Here is a quick example:

Engine engine = new EngineBuilder().build();
ScheduledExecutorService scheduledExecutorService = Executors.newScheduledThreadPool(5);
ScheduledFuture scheduledFuture = scheduledExecutorService.schedule(engine, 5, TimeUnit.SECONDS);

The java.util.concurrent.ScheduledExecutorService is limited in terms of scheduling features. That's why Easy Batch provides an integration module with the popular Java scheduler Quartz.

In this tutorial, you will learn how to schedule a batch job using the BatchJobScheduler API.

This API acts as a bridge between Easy Batch and Quartz to free you from dealing with Quartz's low level APIs.

3. Try it out

First, you need to add the easybatch-quartz module to your project dependencies:

<dependency>
    <groupId>org.easybatch</groupId>
    <artifactId>easybatch-quartz</artifactId>
    <version>3.0.3</version>
</dependency>

Then you can use the BatchJobScheduler API to schedule job executions:

  • At a fixed point of time using scheduleAt(Engine engine, Date when)
  • Repeatedly with predefined interval using scheduleAtWithInterval(Engine engine, Date when, int interval)
  • Using unix cron-like expression with scheduleCron(Engine engine, String cronExpression)

For all details about these APIs, please refer to the Javadoc of the org.easybatch.integration.quartz package.

The listing below shows how to schedule the Hello World tutorial to run every 10 seconds:

public class Launcher {

    public static void main(String[] args) throws Exception {

        // Create the data source
        String dataSource =
            "1,foo,easy batch rocks! #EasyBatch\n" +
            "2,bar,@foo I do confirm :-)";

        // Build a batch engine
        Engine engine = new EngineBuilder()
            .reader(new StringRecordReader(dataSource))
            .processor(new TweetProcessor())
            .build();

        // Schedule the engine to start now and run every 10 seconds
        BatchJobScheduler scheduler = BatchJobScheduler.getInstance();
        scheduler.scheduleAtWithInterval(engine, new Date(), 10);
        scheduler.start();

        System.out.println("Hit enter to stop the application");
        System.in.read();
        scheduler.stop();

    }
}

4. Run the tutorial

4.1 From the command line

To run the tutorial from scratch, follow these steps:

$>git clone https://github.com/EasyBatch/easybatch-tutorials.git
$>cd easybatch-tutorials
$>mvn install
$>mvn exec:java -PrunQuartzTutorial

If everything is ok, you will see in the console that Easy Batch engine will run every 10 seconds.

4.2 From your IDE

First, you need to checkout the source code of the tutorial available here:

$>git clone https://github.com/EasyBatch/easybatch-tutorials.git

If you do not have git installed, you can download a zip file containing the project's source code from GitHub here.

Then, import the easybatch-tutorials module in you favorite IDE and resolve maven dependencies.

Finally, run the org.easybatch.tutorials.advanced.quartz.Launcher class without any argument.

5. Summary

In this tutorial, you have learned how to schedule job executions using either the java.util.concurrent.ScheduledExecutorService or the BatchJobScheduler API.
