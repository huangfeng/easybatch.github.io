Chunk processing Tutorial

1. Introduction

This tutorial is an introduction to the MultiRecord API. A multi-record represents a chunk of records. Easy Batch provides several MultiRecordReaders and MultiRecordWriters to read and write records in chunks.

In this tutorial, you will use the IterableMultiRecordReader to read multi-records from an iterable data source:

List<String> dataSource = Arrays.asList("foo", "bar", "baz", "toto", "titi");

We have a list of persons and we would like to read 2 persons at a time. So Let's get started!

  • JDK 1.6+
  • Maven
  • Git (Optional)

3. Reading persons in chunks

To read persons in chunks, you will use the IterableMultiRecordReader. This reader needs to be configured with an Iterable data source and the chunk size:

List<String> dataSource = Arrays.asList("foo", "bar", "baz", "toto", "titi");

int chunkSize = 2;

Engine engine = new EngineBuilder()
    .reader(new IterableMultiRecordReader<String>(dataSource, chunkSize))
    .build();

This reader will produce MultiRecord instances. Each multi-record has a list of at most 2 records.

4. Processing MultiRecords

To keep the tutorial simple, we will print out the content of each multi-record to inspect which persons it contains. Here is a processor for that:

MultiRecordProcessor.java
public class MultiRecordProcessor implements RecordProcessor<MultiRecord, MultiRecord> {

    @Override
    public MultiRecord processRecord(MultiRecord multiRecord) throws RecordProcessingException {
        System.out.println(multiRecord);
        return multiRecord;
    }
}

5. Writing MultiRecords to a data sink

Just for demonstration purpose, we would like to write persons to a target collection. We can use the CollectionMultiRecordWriter for that:

List<String> dataSource = Arrays.asList("foo", "bar", "baz", "toto", "titi");
List<String> dataSink = new ArrayList<String>();

int chunkSize = 2;

Engine engine = new EngineBuilder()
    .reader(new IterableMultiRecordReader<String>(dataSource, chunkSize))
    .writer(new CollectionMultiRecordWriter(dataSink))
    .build();

The CollectionMultiRecordWriter will write multi-records to the target collection dataSink.

6. Putting it all together

Now that we have a MultiRecordReader, a MultiRecordProcessor and a MultiRecordWriter, let's configure a engine to process the data source:

public class Launcher {

    public static final int CHUNK_SIZE = 2;

    public static void main(String[] args) throws Exception {
        List<String> dataSource = Arrays.asList("foo", "bar", "baz", "toto", "titi");
        List<String> dataSink = new ArrayList<String>();

        EngineBuilder.aNewEngine()
            .reader(new IterableMultiRecordReader<String>(CHUNK_SIZE, dataSource))
            .processor(new MultiRecordProcessor())
            .writer(new CollectionMultiRecordWriter(dataSink))
            .build().call();

        System.out.println("dataSink = " + dataSink);
    }
}

6. Running the tutorial

6.1 From the command line

To run the tutorial from scratch, you can follow these instructions:

$>git clone https://github.com/EasyBatch/easybatch-tutorials.git
$>cd easybatch-tutorials
$>mvn install
$>mvn exec:java -PrunMultiRecordTutorial

You should see something like the following output in the console:

MultiRecord: {
    record = [header=[number=1, source="In-Memory Iterable"], payload="foo"]
    record = [header=[number=2, source="In-Memory Iterable"], payload="bar"]
}
MultiRecord: {
    record = [header=[number=3, source="In-Memory Iterable"], payload="baz"]
    record = [header=[number=4, source="In-Memory Iterable"], payload="toto"]
}
MultiRecord: {
    record = [header=[number=5, source="In-Memory Iterable"], payload="titi"]
}
dataSink = [foo, bar, baz, toto, titi]

As you can see, each multi-record contains 2 persons except the last one which contains only one person since the total number of records is odd.

6.2 From your IDE

First, you need to checkout the source code of the tutorial available here:

$>git clone https://github.com/EasyBatch/easybatch-tutorials.git

If you do not have git installed, you can download a zip file containing the project's source code from GitHub here.

Then, import the easybatch-tutorials module in you favorite IDE and resolve maven dependencies.

Finally, run the org.easybatch.tutorials.intermediate.chunks.Launcher class without any argument.

7. Summary

Congratulation! You have learned how to process records in chunks using a MultiRecordReader and a MultiRecordWriter.
