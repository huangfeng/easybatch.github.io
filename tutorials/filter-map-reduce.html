Filter-Map-Reduce tutorial

1. Introduction

This tutorial is a filter-map-reduce application that calculates some statistics on a list of persons of the following Person type:

public class Person {

    private String name;
    private String country;
    private int age;

    public Person(String name, String country, int age) {
        this.name = name;
        this.country = country;
        this.age = age;
    }

    //getters and setters omitted
}

Here is the list of persons that represent our data source:

List<Person> dataSource = new ArrayList<Person>();
dataSource.add(new Person("jean", "france", 10));
dataSource.add(new Person("foo", "usa", 30));
dataSource.add(new Person("bar", "belgium", 20));
dataSource.add(new Person("jacques", "france", 40));

Let's get started!

3. Example 1: Find the youngest french person's age

In this example, we would like to find the age of youngest french person. First, let's create a engine to read persons from the data source:

Engine engine = EngineBuilder.aNewEngine()
    .reader(new IterableRecordReader<Person>(dataSource))
    .build();

The list of persons is an iterable data source, we can use the IterableRecordReader to read persons from this list. Then, we need to filter non-french persons. Let's create a record filter called CountryFilter :

public class CountryFilter implements RecordFilter {

    private String country;

    public CountryFilter(String country) {
        this.country = country;
    }

    @Override
    public boolean filterRecord(Record record) {
        Person person = (Person) record.getPayload();
        return !country.equalsIgnoreCase(person.getCountry());
    }

}

After that, we need to get the age of each person as the "map" operation. We can create a AgeMapper for that:

public class AgeMapper implements RecordMapper<Integer> {

    @Override
    public Integer mapRecord(Record record) {
        Person person = (Person) record.getPayload();
        return person.getAge();
    }

}

Now, we should calculate the lowest age as the "reduce" operation. Here is a record processor to calculate the minimum age:

public class MinCalculator implements ComputationalRecordProcessor<Integer, Integer, Integer> {

    private Integer min = Integer.MAX_VALUE;

    @Override
    public Integer processRecord(Integer record) {
        if (record < min) {
            min = record;
        }
        return record;
    }

    @Override
    public Integer getComputationResult() {
        return min;
    }

}

Note that this is a ComputationalRecordProcessor since it does some computation. Computational record processors should placed at the end of the pipeline to be able to get their computation result from the execution report:

Engine engine = EngineBuilder.aNewEngine()
    .reader(new IterableRecordReader<Person>(dataSource))
    .filter(new CountryFilter("france"))
    .mapper(new AgeMapper())
    .processor(new MinCalculator())
    .build();

Report report = engine.call();
System.out.println("The youngest french person's age is: " + report.getBatchResult());

The previous listing is the complete application with all components together. If you run this example, you should get the following result:

The youngest french person's age is: 10

3. Example 2: Group persons by country

In this example, we would like to group persons by country. For instance, for the previous data source, we should get the following result:

Persons grouped by country:
{ belgium=[Person{name='bar',country='belgium',age=20}],
  france=[Person{name='jacques',country='france',age=40},Person{name='jean',country='france',age=10}],
  usa=[Person{name='foo',country='usa',age=30}]
}

Since the data source is the same as the previous example, we will reuse the same record reader:

Engine engine = EngineBuilder.aNewEngine()
    .reader(new IterableRecordReader<Person>(dataSource))
    .build();

Then, we need to implement a record processor to group person by country. Here is the code for that:

public class GroupByCountry implements
    ComputationalRecordProcessor<Record, Record, Map<String, Set<Person>>> {

    private  Map<String, Set<Person>> personsByCountry = new HashMap<String, Set<Person>>();

    @Override
    public Record processRecord(Record record) {
        Person person = (Person) record.getPayload();
        String country = person.getCountry();
        if (!personsByCountry.containsKey(country)) {
            personsByCountry.put(country, new HashSet<Person>());
        }
        personsByCountry.get(country).add(person);
        return record;
    }

    @Override
    public Map<String, Set<Person>> getComputationResult() {
        return personsByCountry;
    }

}

This computational record processor groups persons by country in a java.util.Map instance. That's all what we need for this example, let's register the GroupByCountry processor:

Engine engine = EngineBuilder.aNewEngine()
    .reader(new IterableRecordReader<Person>(dataSource))
    .processor(new GroupByCountry())
    .build();

If you run this example, you should get the following result:

Persons grouped by country:
{ belgium=[Person{name='bar',country='belgium',age=20}],
  france=[Person{name='jacques',country='france',age=40},Person{name='jean',country='france',age=10}],
  usa=[Person{name='foo',country='usa',age=30}]
}

4. Running the tutorial

4.1 From the command line

To run the tutorial from scratch, you can follow these instructions:

$>git clone https://github.com/EasyBatch/easybatch-tutorials.git
$>cd easybatch-tutorials
$>mvn install
$>mvn exec:java -PrunFilterMapReduceTutorial

4.2 From your IDE

First, you need to checkout the source code of the tutorial available here:

$>git clone https://github.com/EasyBatch/easybatch-tutorials.git

If you do not have git installed, you can download a zip file containing the project's source code from GitHub here.

Then, import the easybatch-tutorials module in you favorite IDE and resolve maven dependencies.

Finally, run the org.easybatch.tutorials.basic.filterMapReduce.Launcher class without any argument.

5. Summary

Congratulations! You have built a Filter-Map-Reduce application with Easy Batch.
